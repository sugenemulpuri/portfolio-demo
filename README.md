# Sugene's Portfolio

This is a demo for the portfolio page that will be featured on Sugene Mulpuri's website launching in December 2025. Content is subject to change.

## Description

This portfolio features 4 projects completed by Sugene throughout his career. It also includes contact information and a visitor submission form.

## Getting Started

Hover over each project section to view the project number and description. You can also navigate to the Home and About pages once the full website is live.

### Dependencies

- HTML5
- CSS3

### Installing

The `CS601_HW3_Mulpuri.zip` file contains all necessary files including HTML, CSS, and images.

To run the page:
1. Download and unzip CS601_HW3_Mulpuri.zip
2. Open index.html in your web browser
